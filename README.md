## 🎯 Project Overview

This system predicts house prices based on input features like median income, house age, average rooms, and location coordinates. The predictions are:
- Generated by a trained Random Forest regression model
- Saved to a MySQL database for historical tracking
- Served through a REST API
- Displayed in a responsive React frontend

## 🚀 Setup Instructions

### 1. Machine Learning Pipeline Setup

Train the ML model to generate the pickle file needed by the backend.

#### Step 1a: Navigate to ML Pipeline Directory
```powershell
cd d:\HousePrice\ml-pipeline
```

#### Step 1b: Create Virtual Environment
```powershell
python -m venv venv
.\venv\Scripts\Activate.ps1
```

#### Step 1c: Install Dependencies
```powershell
pip install -r requirements.txt
```

#### Step 1d: Train the Model
```powershell
python src/train_model.py
```

**Output**: Creates `models/house_price_pipeline.pkl` (~10-20 MB)

**Model Details**:
- Algorithm: Random Forest Regressor (100 trees)
- Input Features: 8 (MedInc, HouseAge, AveRooms, AveBedrms, Population, AveOccup, Latitude, Longitude)
- Dataset: California Housing Dataset
- Train/Test Split: 80/20
- Preprocessing: StandardScaler

## 📚 API Documentation

### Base URL
```
http://localhost:8000
```

### Interactive API Docs
Visit `http://localhost:8000/docs` for Swagger UI

### Endpoints

#### 1. **POST /predict**
Predict house price and save to database.

**Request Body**:
```json
{
  "MedInc": 8.3252,
  "HouseAge": 41.0,
  "AveRooms": 6.984127,
  "AveBedrms": 1.023638,
  "Population": 322.0,
  "AveOccup": 2.555556,
  "Latitude": 37.88,
  "Longitude": -122.23
}
```

**Response** (200 OK):
```json
{
  "predicted_price": 4.536,
  "message": "Prediction saved to database",
  "record_id": 1
}
```

**Error Response** (422 Unprocessable Entity):
- Missing or invalid input fields
- Data type mismatch

## 🤖 Model Details

### Training Script: `ml-pipeline/src/train_model.py`

**Algorithm**: Random Forest Regressor
- **Trees**: 100
- **Max Depth**: 15
- **Random State**: 42 (reproducible)

**Input Features** (8 features):
1. Median Income (MedInc)
2. House Age
3. Average Rooms
4. Average Bedrooms
5. Population
6. Average Occupancy
7. Latitude
8. Longitude

**Dataset**: California Housing Dataset (20,640 samples)
- **Train Set**: 16,512 samples (80%)
- **Test Set**: 4,128 samples (20%)

**Preprocessing**: StandardScaler (mean=0, std=1)

**Metrics Reported**:
- Mean Absolute Error (MAE)
- R² Score

**Output**: `models/house_price_pipeline.pkl`
- Includes both scaler and model
- Serialized using joblib
- Auto-scales input before prediction

## 🐛 Troubleshooting

### Backend Won't Start
```
Error: ModuleNotFoundError: No module named 'fastapi'
```
**Solution**: Ensure virtual environment is activated and dependencies installed:
```powershell
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

### Frontend Can't Connect to Backend
```
Error: Failed to fetch from http://localhost:8000
```
**Solutions**:
1. Ensure backend is running on port 8000
2. Check CORS configuration in `backend/main.py`
3. Update API_URL in `frontend/src/App.jsx` if using different port

### Database Connection Error
```
Error: Can't connect to MySQL server
```
**Solutions**:
1. Verify MySQL server is running
2. Check connection credentials in `backend/db.py`
3. Ensure database exists: `sanket_house_price_db`

### ML Model Not Found
```
Error: FileNotFoundError: house_price_pipeline.pkl
```
**Solution**: Train the model first:
```powershell
cd ml-pipeline
python src/train_model.py
```
